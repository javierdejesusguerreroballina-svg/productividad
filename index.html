¿Cómo funciona? Elige el modo que se adapte a lo que necesitas hacer ahora. Rush para tareas urgentes con tiempo limitado. Flow para trabajo profundo y creativo. Reunión para gestionar juntas con foco y registro.

Gestión de Juntas y Control.

Modo Rush

Para tareas urgentes con entrega definida. El timer cuenta regresivo con presión sonora y mensajes de activación. Al 80% del tiempo restante, el modo urgente se activa.

Modo Flow

Para trabajo profundo que requiere concentración sostenida. Ruido marrón de fondo para facilitar el estado de flujo. Sin interrupciones, sin urgencia.

Modo Reunión

Gestiona juntas con timer y registro de atención. Registra distracciones y aportes para ver tu patrón de participación al terminar.
