# javierwarrior.com — Landing Page

Sitio web personal de Javier Guerrero Ballina.  
Psicología aplicada y neurociencia accesible.

## Descripción

Landing page personal construida en HTML puro, sin frameworks ni dependencias.  
Desplegada en Cloudflare Pages, conectada a este repositorio vía GitHub.

## Estructura

```
/
├── index.html       # Página principal
├── profile.png      # Foto de perfil
├── README.md        # Este archivo
├── LICENSE          # Licencia del proyecto
└── .gitignore       # Archivos ignorados por Git
```

## Tecnologías

- HTML5 / CSS3 puro
- Google Fonts (Playfair Display + Lato)
- Cloudflare Pages (hosting y CDN)

## Despliegue

Cada push a la rama `main` despliega automáticamente en [javierwarrior.com](https://javierwarrior.com) vía Cloudflare Pages.
